Career Recommendation Mapping

Create and manage career recommendations for each MBTI type.

Admin Guidelines (Click to expand)
  1. Filter (MBTI Type)
    Select a type (e.g., ENFJ) and click Apply to focus on one MBTI group while adding or reviewing careers. Choose All to manage everything in one list.
  2. Required fields
    You must fill in: Type, Career Key, Career Name, URL, English Title, and English Career Description.
  3. Career Key format (stable & unique)
    Use snake_case only. Example: ux_researcher, data_analyst.
    Allowed: a–z, 0–9, _. No spaces, no special symbols.
    Tip: Don’t rename the key unless necessary (it links translations).
  4. URL rules
    Must start with https:// and should be a reliable page (official / O*NET / JobStreet / university / gov).
    Avoid unstable links (short links, login-required pages).
  5. Title & description quality
    Keep descriptions short: 1–2 sentences. Explain why the career fits the MBTI type (skills/style).
    Avoid generic text like “good job” or “nice career”.
  6. Translation (EN → BM/ZH)
    Write EN first, then click Translate to generate BM and ZH.
    After generating, quickly review BM/ZH for accuracy (names and terms may need manual edits).
  7. Delete carefully
    Deleting is permanent. Confirm you selected the correct MBTI type and Career Key before deleting.
